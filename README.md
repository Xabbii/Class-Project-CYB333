# Class Project CYB333

##Password Strength Analyzer

Project Description:

My name is Javier, and this repository is for my CYB333 class project. For this project, I will be creating a Python password strength analyzer.

Purpose of the Project:

The purpose of this project is to build a tool that checks how strong a password is. The program will review passwords based on length, uppercase letters, lowercase letters, numbers, special characters, and common weak password patterns

Goal:

The Goal of the password analyzer is to help users understand why a password may be weak, medium, or strong. The program will also provide suggestions on how the user can improve the password.
